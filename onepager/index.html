This type of grading adheres to the conventional green and red flagging. It matches the student answer to the answer key exactly.

Answers that are near misses or synonyms are flagged as orange for the teacher to notice. They are still graded as wrong unless the teacher adds them to the answer key. This feature employs linguistic similarity in context with modifiable threshold regardless of the language in question.

When this feature is activated orange answers are automatically scored as green. This could result in false scoring such as 'smar' for 'smart'.

This feature is part of the support the teacher receives during the creation of exercises. It attempts to identify, classify, and highlight existing grammar structures in the text. It also attempts to average all the structure levels into a unified label. The source of the maping is CEFR-j.

This feature is part of the support the teacher receives during the creation of exercises. It attempts to identify the vocabulary level. It is based on CEFR-j and Octanove.
